Helpdesk Ticket Management System
A web application for managing support requests from submission to resolution. Support agents handle all tickets in one place, with full communication history and real-time status tracking.
Key features:

ticket creation, assignment and prioritization
categories and status workflow (new → in progress → waiting → closed)
in-ticket messaging thread between agent and user
email/push notifications
response time reports and agent performance stats
